# remark-directive: полная шпаргалка

Синтаксис: **Generic Directives (proposal для CommonMark)**

Директивы позволяют добавлять **кастомные элементы Markdown**, которые
затем можно преобразовывать в HTML, React-компоненты, ProseMirror-ноды,
MDX-компоненты и т.п.

Используются в экосистеме:

-   remark
-   mdast
-   MDX
-   документационные сайты
-   кастомные Markdown-редакторы

------------------------------------------------------------------------

# Типы директив

В спецификации существует **три типа директив**.

  -------------------------------------------------------------------------------------
  Тип               AST тип              Синтаксис                    Назначение
  ----------------- -------------------- ---------------------------- -----------------
  Text              textDirective        `:abbr[HTML]{title="..."}`   inline директива

  Leaf              leafDirective        `::youtube{id="abc"}`        блочная директива
                                                                      без контента

  Container         containerDirective   `:::alert{type="warning"}`   блочная директива
                                                                      с контентом
  -------------------------------------------------------------------------------------

------------------------------------------------------------------------

# Общая структура директивы

    :fenceName[label]{attributes}

Компоненты:

  Часть           Обязательность   Пример
  --------------- ---------------- ------------------
  двоеточия       обязательно      `:`, `::`, `:::`
  имя директивы   обязательно      `alert`
  label           опционально      `[text]`
  attributes      опционально      `{key="value"}`

------------------------------------------------------------------------

# Количество двоеточий

  Тип         Минимум
  ----------- ---------
  text        `:`
  leaf        `::`
  container   `:::`

Особенности:

-   максимального ограничения нет
-   можно использовать любое количество
-   закрывающий fence должен содержать **то же количество двоеточий**

------------------------------------------------------------------------

# Вложенность

Работает правило:

**больше двоеточий = более внешний контейнер**

Пример:

    :::::section
    ::::block
    :::alert

------------------------------------------------------------------------

# Правило закрытия

Закрывающий fence должен содержать **ровно столько же двоеточий**,
сколько у контейнера.

  Открыли          Закрываем
  ---------------- -----------
  `:::alert`       `:::`
  `::::block`      `::::`
  `:::::section`   `:::::`

------------------------------------------------------------------------

# Пустая строка перед закрытием

Перед закрывающим fence **обязательно должна быть пустая строка**.

Правильно:

    :::alert
    Текст

    :::

Неправильно:

    :::alert
    Текст
    :::

------------------------------------------------------------------------

# Text Directive (inline)

Inline-директива используется **внутри текста**.

## Синтаксис

    :name[label]{attrs}

## Пример

    Hover :tooltip[text]{content="Hello"}

## AST

    textDirective
      name: tooltip
      attributes:
        content: Hello
      children:
        text "text"

Особенности:

-   inline
-   может иметь children
-   используется внутри paragraph
-   label может содержать markdown

------------------------------------------------------------------------

# Leaf Directive (block без children)

Leaf-директива --- это **блочный элемент без содержимого**.

## Синтаксис

    ::name{attrs}

## Примеры

    ::youtube{id="abc123"}
    ::productSlider{query="phones"}
    ::diagram{src="schema.mmd"}

AST:

    leafDirective
      name: youtube
      attributes:
        id: abc123

Особенности:

-   block
-   **нет children**
-   используется для embed-виджетов

------------------------------------------------------------------------

# Container Directive (block с children)

Container-директива --- это **блочный контейнер с содержимым**.

## Синтаксис

    :::name{attrs}
    content
    :::

## Пример

    :::alert{type="warning"}
    Be careful
    :::

AST:

    containerDirective
      name: alert
      attributes:
        type: warning
      children:
        paragraph

Особенности:

-   block
-   содержит children
-   может содержать любой markdown

------------------------------------------------------------------------

# Атрибуты

## Именованные

    {key="value"}

Пример:

    ::alert{type="warning"}

## Классы

    {.class}

эквивалентно

    {class="class"}

## ID

    {#id}

эквивалентно

    {id="id"}

## Комбинация

    {#id .class a="b"}

## Объединение классов

    {.red class=green .blue}

результат:

    class="red green blue"

------------------------------------------------------------------------

# Имя директивы

Правила:

-   первый символ --- буква
-   допустимы буквы, цифры, `-`, `_`
-   не может заканчиваться на `-` или `_`

Примеры:

    alert
    callout-box
    my_widget

------------------------------------------------------------------------

# Label

Особенности:

-   опционален
-   `:x` и `:x[]` эквивалентны
-   может содержать markdown
-   в leaf и container **не допускаются переносы строк**

Пример:

    :tooltip[*hover me*]{text="Hello"}

------------------------------------------------------------------------

# Ограничения

-   label до \~999 символов
-   до 32 уровней вложенных скобок

------------------------------------------------------------------------

# Особенность text directive

Конструкция

    :red:

не работает (конфликт с emoji).

Нужно писать:

    :red[]

или

    :red{}

------------------------------------------------------------------------

# Запрещённые пробелы

Нельзя писать:

    : a
    :a []
    :a {}
    :a[] {}

------------------------------------------------------------------------

# Если закрытие отсутствует

Если closing fence не найден, контейнер продолжается до конца родителя:

-   документа
-   blockquote
-   list item

------------------------------------------------------------------------

# Пример полного документа

    # Guide

    Some text

    :::alert{type="info"}
    Important message
    :::

    Hover :tooltip[text]{content="Hello"}

    ::youtube{id="abc123"}
